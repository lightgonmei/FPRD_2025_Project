    # Why Virtual environment? (Tabui zeang virtual environment kasiam tong cho?)
    > Any system akhat tuang default python installed version hei latest python version bam gei nu e. 
    > Mi latest python version tho ML (machine learning) tong libraries geitho work sw mak bam nu e, the best practice to use is under or below, python version 3.12.
    > System akhat kho python version hei latest sijin bam nu de, tiki mita tho ML project gei tho work sw chui lk ni pui zang khou rui, work sw nimei version hap laki sijin numei hei, Virtual environment hap laki sijin nu khang the. 
    > Virtual environment tumei hei system akhat akum tong e, specific work taanc khang.
    > E.G. Python version 3.10 hei system kho install sw kei lou dat the turakho, mi tihei tabui thiak sijin khang cho tu e thei, virtual environment create/ kasiam puigan kho, py -version 3.10 tong path ta add swkhang the eg:- [python3.10 -m venv environment_name].
